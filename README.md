# 📚 Book Data Extractor

A Python-based web scraping project that extracts book information from HTML files using BeautifulSoup and stores the extracted data in CSV format for further analysis.

---

## 🚀 Project Overview

This project demonstrates the fundamentals of web scraping and data processing using Python.

The program:

* Loads book data from HTML files
* Extracts Book Title, Price, and Rating
* Stores the extracted information in a Pandas DataFrame
* Exports the cleaned data into a CSV file

This project helped me practice:

* Web Scraping
* HTML Parsing
* Data Extraction
* Data Formatting
* CSV File Handling
* Object-Oriented Programming (OOP)

---

## 🛠 Technologies Used

* Python 3
* BeautifulSoup4
* Pandas

## ⚙️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/book-data-extractor.git
```

### 2. Navigate to the Project Directory

```bash
cd book-data-extractor
```

### 3. Install Required Libraries

```bash
pip install pandas beautifulsoup4
```

### 4. Run the Program

```bash
python "Book Data Extractor.py"
```

### 5. Enter HTML File Name

```text
Enter HTML file name: books
```

### 6. Enter Output File Name

```text
Enter file name to save data: all_books
```

### 7. Output

The extracted data will be saved as:

```text
all_books.csv
```
